# imbcadeiadevalor

Estrutura organizada para trabalhar no material de POPs da IMB:

- `imb_pops_iso9001.html`: documento principal e ponto de entrada.
- `assets/css/imb-pops.css`: estilos extraídos do HTML original.
- `assets/js/imb-pops-data.js`: base de dados dos 24 POPs.
- `assets/js/imb-pops-app.js`: lógica de filtros, busca e expansão da interface.

Para editar o conteúdo:

- altere layout e marcação no HTML principal;
- altere estilos no arquivo CSS;
- altere os POPs no arquivo de dados;
- altere comportamento no arquivo JavaScript da aplicação.
